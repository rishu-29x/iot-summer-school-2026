## 📖 About This Repository

This repository documents my complete journey through the **IoT Summer School 2026** — including weekly assignments, theory notes, project documentation, and mini-projects built along the way. It reflects hands-on work with Arduino, ESP32, embedded systems, communication protocols, IoT networking, sensor interfacing, and version control using Git/GitHub.

Everything is organized week-by-week, with each folder containing the relevant source code, circuit diagrams, notes, sample outputs, and a project-specific README.
