# Q12. Microcontroller vs Microprocessor

| **Parameter**         | **Microcontroller (Arduino UNO)**                           | **Microprocessor (Raspberry Pi)**                                  |
| --------------------- | ----------------------------------------------------------- | ------------------------------------------------------------------ |
| **CPU Speed**         | 16 MHz                                                      | 1–2+ GHz (depends on Raspberry Pi model)                           |
| **On-chip RAM**       | 2 KB SRAM                                                   | Typically 1 GB–8 GB external RAM                                   |
| **On-chip Flash**     | 32 KB Flash Memory                                          | No on-chip flash; uses a microSD card for storage                  |
| **Operating System**  | No operating system (runs a single program directly)        | Runs an operating system such as Raspberry Pi OS (Linux)           |
| **Typical Use Case**  | Embedded systems, sensor interfacing, robotics, IoT control | General-purpose computing, multimedia, AI, IoT gateway, web server |
| **Power Consumption** | Low (typically around 50–70 mA at 5 V)                      | Higher (typically 2–7 W depending on the model and workload)       |
